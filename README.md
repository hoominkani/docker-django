Django applications with Docker
=============

## tl;dr
```bash
$ git clone git@github.com:hoominkani/docker-django.git
$ docker-compose up
```

You can visit your app at<https://localhost> and the admin site
at <https://localhost/admin>.

Stack and version numbers used:

| Name           | Version  |
|----------------|----------|
| Django         | 2.1.8    |
| Nginx          | 1.15     |
| Postgresql     | 11.2     |
